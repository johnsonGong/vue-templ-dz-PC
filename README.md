# vue-templ-dz-PC
任职cqdz开发框架之 模板: PC端 -- 鉴于 VUE CLI 3.X已经发布，此模板不再更新维护。

## 简介
- 1. 增加状态管理组件: vuex;
- 2. 增加ajax组件: axios, qs;
- 3. 增加本地Mock方案: json-server, 路径: mock;
- 4. 加入 api 调用方案;
- 5. 加入 less 包;

## 使用此模板生成开发框架
``` bash
vue init username/repo my-project

vue init johnsonGong/vue-templ-dz-PC dz-demo-pc
```
